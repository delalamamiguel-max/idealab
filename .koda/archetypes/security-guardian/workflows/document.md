# Document: Security Guardian

## Goal

Produce a shareable security packet from the evidence pack.

## Steps

1. Confirm `security/evidence/` is complete.
2. Create `security/security-packet.md` containing:

- Scope and system context
- Risk profile and verification target
- Threat model summary (top threats + mitigations)
- Secure-by-default summary
- Security test coverage summary
- Vulnerability management summary
- Metrics snapshot

3. Link to evidence artifacts and SBOM locations.

## Success criteria

- `security/security-packet.md` exists and links to evidence artifacts.
