# Security Metrics

## Goals

Define the minimal metrics that demonstrate security posture trends.

## Metrics

### Delivery / prevention

- % of repos with evidence pack present
- % of releases with SBOM recorded
- % of projects with threat model updated in last N months

### Vulnerability management

- Open vulnerabilities by severity
- Mean time to remediate by severity
- SLA breach count

### Operational security

- Incidents by severity
- Repeated incident categories

## Reporting cadence

- Weekly: vulnerability aging + SLA breaches
- Monthly: posture summary and trendline
