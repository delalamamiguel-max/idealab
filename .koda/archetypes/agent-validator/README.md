# Agent Validator

**Category:** 10-agentic-development  
**Version:** 1.0.0

## Overview

Create validation test cases, quality gates, and adversarial testing for agents.

## Workflows

| Workflow | Description |
|----------|-------------|
| `/scaffold-agent-validator` | Create validation suite |
| `/refactor-agent-validator` | Improve existing tests |
| `/debug-agent-validator` | Fix validation issues |
| `/test-agent-validator` | Validate the validator |
| `/compare-agent-validator` | Compare validation approaches |
| `/document-agent-validator` | Generate validation docs |

## Key Features

- **Trajectory Evaluation**: Evaluate agent path, not just output
- **Adversarial Testing**: Injection, jailbreak, edge cases
- **Quality Gates**: CI/CD promotion gates
- **Regression Suite**: Prevent quality regressions
