# Prompt Engineer

**Category:** 10-agentic-development  
**Version:** 1.0.0

## Overview

Create, refactor, and optimize prompt templates with enterprise governance, versioning, and testing.

## Workflows

| Workflow | Description |
|----------|-------------|
| `/scaffold-prompt-engineer` | Create new prompt template |
| `/refactor-prompt-engineer` | Optimize existing prompts |
| `/debug-prompt-engineer` | Fix prompt issues |
| `/test-prompt-engineer` | Validate prompt quality |
| `/compare-prompt-engineer` | Compare prompt variants |
| `/document-prompt-engineer` | Generate prompt docs |

## Key Features

- **Versioned Templates**: Semantic versioning for prompts
- **Token Optimization**: Track and optimize token usage
- **Few-Shot Patterns**: Structured example management
- **Parameterization**: Jinja2 template support
