# MCP Developer

**Category:** 10-agentic-development  
**Version:** 1.0.0

## Overview

Build Model Context Protocol (MCP) tool servers with proper schema validation, error handling, and security boundaries.

## Workflows

| Workflow | Description |
|----------|-------------|
| `/scaffold-mcp-developer` | Create new MCP tool server |
| `/refactor-mcp-developer` | Improve existing tools |
| `/debug-mcp-developer` | Fix tool errors |
| `/test-mcp-developer` | Validate tool correctness |
| `/compare-mcp-developer` | Compare tool approaches |
| `/document-mcp-developer` | Generate tool docs |

## Key Features

- **Schema Validation**: Pydantic models for all inputs/outputs
- **Structured Errors**: Consistent error response format
- **Security Boundaries**: Least privilege, input sanitization
- **Tool Correctness**: Evaluation metrics integration
