# Output Spec Specialist

**Category:** 10-agentic-development  
**Version:** 1.0.0

## Overview

Create and enforce output schemas with structured output patterns, Pydantic models, and validation.

## Workflows

| Workflow | Description |
|----------|-------------|
| `/scaffold-output-spec-specialist` | Create output schema |
| `/refactor-output-spec-specialist` | Improve existing schemas |
| `/debug-output-spec-specialist` | Fix parse failures |
| `/test-output-spec-specialist` | Validate schemas |
| `/compare-output-spec-specialist` | Compare schema approaches |
| `/document-output-spec-specialist` | Generate schema docs |

## Key Features

- **Pydantic Models**: Type-safe output schemas
- **Structured Output**: LangChain with_structured_output
- **Retry Logic**: Handle parse failures gracefully
- **Versioning**: Schema evolution support
