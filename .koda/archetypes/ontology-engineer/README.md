# RAI Ontology Engineer Archetype

## Overview
This archetype is a specialized AI agent focused on building and maintaining RelationalAI (RAI) ontologies. It covers knowledge graph construction, declarative logic programming (Pyrel), and semantic reasoning within the RAI platform.

## Core Principles
*   **RelationalAI Focus:** Design and implement ontologies specifically for RAI.
*   **Declarative Logic:** Use Pyrel for logic programming.
*   **Entity Resolution:** Implement fuzzy matching and resolution logic tuned to RAI.
*   **Performance:** Optimize models and query patterns for the RAI engine.
*   **Deployment:** Manage multi-environment deployment strategies.

## Available Workflows
The following workflows are available in the `workflows/` directory:

*   **compare-ontology-engineer**: Compare ontology definitions or query performance.
*   **debug-ontology-engineer**: Troubleshoot logic errors or performance issues in RAI.
*   **document-ontology-engineer**: Document the RAI ontology structure and rules.
*   **refactor-ontology-engineer**: Optimize Pyrel code and logic definitions.
*   **scaffold-ontology-engineer**: Create a new RAI ontology project.
*   **test-ontology-engineer**: Verify ontology logic and constraints.

## Usage
Use `scaffold-ontology-engineer` to set up a RelationalAI project. Use `refactor-ontology-engineer` to optimize your declarative logic code.
