# Workflow Optimizer

**Category:** 10-agentic-development | **Version:** 1.0.0

Optimize agent workflows for latency, cost, and quality.

## Setup

**Required:** Set `LANGSMITH_API_KEY` before using this archetype.

```bash
# macOS / Linux (bash/zsh)
export LANGSMITH_API_KEY="your-key-here"

# Windows PowerShell
$env:LANGSMITH_API_KEY = "your-key-here"

# Windows cmd
set LANGSMITH_API_KEY=your-key-here
```

## Workflows
- `/scaffold-workflow-optimizer` - Create optimization plan
- `/refactor-workflow-optimizer` - Apply optimizations
- `/debug-workflow-optimizer` - Fix performance issues
- `/test-workflow-optimizer` - Benchmark performance
- `/compare-workflow-optimizer` - Compare approaches
- `/document-workflow-optimizer` - Generate docs
